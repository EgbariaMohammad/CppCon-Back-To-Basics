## About

This collection represents CppCon's "Back to Basics" track, where expert instructors teach fundamental C++ concepts from first principles. Whether you're a beginner or an experienced developer, these talks provide valuable insights into modern C++ best practices.

**Track Chairs:** Mike Shah & Klaus Iglberger

**Total Content:** 6 years | 82 talks | ~82 hours of C++ fundamentals
